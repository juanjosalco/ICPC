## 📈 Study Roadmap

### Phase 1: Foundation (→ Specialist ~1400)
- CSES Introductory ✅ + Sorting & Searching
- CF EDU: Binary Search, Two Pointers
- Compete in every Div 3/4 round
- CP Algorithms: Basic data structures

### Phase 2: Intermediate (→ Expert ~1600)
- CSES Dynamic Programming + Graphs
- CF EDU: Segment Tree Part 1, DSU
- Start Div 2 contests, begin 2h virtual simulations
- CP Algorithms: Graph theory, DP

### Phase 3: Advanced (→ Candidate Master ~1900)
- CSES Range Queries + Trees + Strings
- CF EDU: Segment Tree Part 2, Suffix Array, Z-Function
- Start 5h ICPC simulations with team
- CP Algorithms: Flows, matching, advanced DP

### Phase 4: Mastery (→ Master 1900+)
- CSES Math, Geometry, Advanced Techniques, Additional
- CP Algorithms: Geometry, combinatorics, number theory
- Regular team simulations + post-analysis
- Upsolve every contest problem
