## Study Order (Recommended)
1. ✅ Binary Search → foundation for everything
2. Two Pointers → complements binary search
3. DSU → needed for graph algorithms
4. Segment Tree Part 1 → core data structure
5. Segment Tree Part 2 → advanced applications
6. Z-Function → string processing foundation
7. Suffix Array → advanced string processing

---

## Tips
- Complete **all steps** in each course before moving to the next
- After each step, solve 2-3 related problems from CSES or Codeforces
- Implement the data structure from scratch at least once without reference
- Add your solutions as `Step{N}_{problem_letter}.cpp` (e.g., `Step1_A.cpp`)
